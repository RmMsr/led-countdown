LED countdown
=============

A 6 x 10 LED matrix controlled via an Arduino as a wall mount countdown.

Parts
-----

The LEDs consist of RGB LED stripes with a chainable led controller.

  * 60 NeoPixel RGB LEDs
  * Pushbutton
  * Arduino compatible controller
  * 5 V wall plug
  * A box (paper)
